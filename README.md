# ДОКУМЕНТАЦИЯ ПОЛЬЗОВАТЕЛЯ

## Цель: извлечения квадратного корня.

## Задачи:
- Изменения языка;
- Вывод результат извлечения квадратного корня

## Преимущество:
- Возможность локализация на 4 языках: Русский, Английский, Испанский, Китайский;
- Возможность вычисления корня с любой степенью;
- Возможность извлекать корни из целых, вещественных и отрицательных чисел;
- Возможность получить результат не только в целом, но и вещественном виде

## Элементы пользовательского интерфейса:
1. Поле ввода числа. Ввод числа, из которого вычисляется корень.
2. Поле ввода степени. Окно ввода степени.
3. Кнопка «Вычислить». После ввода числа и степени, при нажатии кнопки, выводится результат.
4. Выпадающий список выбора языка. При нажатии, появляется список выбора одного из четырех языков.

## Типовые задачи (Сценарии использования)

### Сценарий 1: Вычисление квадратного корня из 81
1. Действие: В поле "Поле ввода числа" введите 81.
2. Действие: В поле "Поле ввода степени" убедитесь, что указано 2 (значение по умолчанию для квадратного корня).
3. Действие: Нажмите кнопку «Вычислить».
4. Результат: В поле результата отобразится 9.

### Сценарий 2: Вычисление кубического корня из -27
1. Действие: В поле "Поле ввода числа" введите -27.
2. Действие: В поле "Поле ввода степени" введите 3.
3. Действие: Нажмите кнопку «Вычислить».
4. Результат: В поле результата отобразится -3.

### Сценарий 3: Вычисление корня 4-й степени из 16 с точным результатом
1. Действие: В поле "Поле ввода числа" введите 16.
2. Действие: В поле "Поле ввода степени" введите 4.
3. Действие: Нажмите кнопку «Вычислить».
4. Результат: В поле результата отобразится 2.

### Сценарий 4: Смена языка интерфейса на английский
1. Действие: Найдите на экране "Выпадающий список выбора языка". По умолчанию в нем, вероятно, установлен "Русский".
2. Действие: Нажмите на список, чтобы раскрыть варианты.
3. Действие: Выберите из списка опцию English.
4. Результат: Все надписи в интерфейсе (заголовки полей, текст кнопки) мгновенно изменятся на английский язык.

## Типовые проблемы и их решения

### Приложение выдает ошибку или не вычисляет корень.
**Возможная причина:**
1. Введено недопустимое значение (например, текст).
2. Попытка извлечь корень четной степени из отрицательного числа (например, квадратный корень из -4).

**Решение:**
1. Убедитесь, что в полях ввода только числа. Разделитель дробной части — точка (.) или запятая (,).
2. Запомните: корень четной степени из отрицательного числа является комплексным числом и в этом приложении не вычисляется. Используйте нечетную степень для отрицательных чисел.

### Не могу найти кнопку вычисления.
**Возможная причина:**
Интерфейс переведен на незнакомый язык.

**Решение:**
Используйте "Выпадающий список выбора языка", чтобы переключиться на знакомый вам язык. Кнопка обычно расположена под полями ввода.
